# golang-cli-dolt-connect

## Description
A POC for connecting a database to golang.

## Tech stack
- bash
- dolt
- golang

## Docker stack
- ubuntu:latest
- dolthub/dolt-sql-server

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
